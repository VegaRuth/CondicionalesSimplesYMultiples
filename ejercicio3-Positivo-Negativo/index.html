Ejercicio #3: POSITIVO O NEGATIVO

Realice un algoritmo para determinar si un número es positivo o negativo. Represéntelo en pseudocódigo y diagrama de flujo

Pseudocódigo

Variables
Nombre Descripción Tipo
num Primer valor a evaluar float

Proceso

  1. Inicio
  2. Leer "num"
  3. Condición

    Si "num" es < 0 entonces alerta "numero negativo";

    Si no "num" es > 0, entonces alerta "numero positivo";

  4. Fin
