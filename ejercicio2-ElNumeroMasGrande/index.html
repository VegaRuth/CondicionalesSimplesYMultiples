Ejercicio #2: ¿CUÁL ES EL NÚMERO MAS GRANDE?

Se desea implementar un algoritmo para determinar cuál de dos valores proporcionados es el mayor. Representarlo con pseudocódigo y diagrama de flujo

Pseudocódigo

Variables
Nombre Descripción Tipo
n1 Primer valor a comparar Int
n2 Segundo valor a comparar Int


Proceso

  1. Inicio
  2. Leer "n1" y "n2"
  3. Condición

    Si "n1" es < "n2", entonces alerta "es menor";

    Y si "n1" es > "n2", entonces alerta "es mayor";

    Si no, entonces: alerta "es igual";
  4. Fin
